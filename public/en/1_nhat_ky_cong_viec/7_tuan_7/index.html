1.7. Week 7 Worklog

Week Objectives:

  • Thiết kế giao diện tổng quan và cấu trúc của Website.
  • Xây dựng kho lưu trữ chung và quy trình làm việc nhóm qua GitHub.
  • Thiết lập môi trường phát triển và kết nối cơ sở dữ liệu.
  • Thử nghiệm triển khai sớm (Deploy) các thành phần hệ thống lên AWS.

Day Task Start Date End Date References

- Đẩy (Push) code khởi tạo lên GitHub		 03/06/2026 03/06/2026

Week Results:

  • Đã hoàn thành bản thiết kế cấu trúc và giao diện cho website, giúp các lập trình viên có hướng đi rõ ràng.
  • Xây dựng thành công môi trường quản lý mã nguồn chung với GitHub, tạo nền tảng vững chắc để làm việc nhóm.
  • Khởi tạo thành công các thành phần cốt lõi của ứng dụng (Frontend, Backend, DB) và thiết lập môi trường triển khai cơ bản.
  • Hoàn thành thiết kế kiến trúc hệ thống trên AWS và triển khai thử nghiệm bước đầu thành công.
  • Chuẩn bị đầy đủ cơ sở hạ tầng, môi trường và source code để phục vụ cho giai đoạn code tính năng và triển khai ứng dụng thực tế.
