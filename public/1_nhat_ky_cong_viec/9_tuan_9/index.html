1.9. Worklog Tuần 9

Mục tiêu tuần 9:

  • Phát triển hoàn thiện các tính năng cốt lõi như giỏ hàng, quản lý sản phẩm.
  • Khớp nối thành công luồng dữ liệu (API) giữa Frontend và Backend.
  • Tối ưu hóa cấu hình hệ thống trên AWS.
  • Triển khai website lên môi trường Cloud và thực hiện kiểm thử (Testing) toàn diện.

Các công việc cần triển khai trong tuần này:

Thứ Công việc Ngày bắt đầu Ngày hoàn thành Nguồn tài liệu
2 - Lập trình logic chức năng quản lý sản phẩm (Thêm/Sửa/Xóa)
- Phát triển chức năng Giỏ hàng (Add to Cart, Checkout)		 15/06/2026 15/06/2026
3 - Tích hợp luồng truyền tải dữ liệu giữa Frontend và Backend thông qua API
- Xử lý lỗi liên quan đến kết nối và đồng bộ dữ liệu		 16/06/2026 16/06/2026
4 - Rà soát, kiểm tra và tối ưu cấu hình hạ tầng AWS (Security Group, IAM Role, S3 Bucket Policy) để đảm bảo hệ thống an toàn và tối ưu tài nguyên 17/06/2026 17/06/2026
5 - Triển khai (Deploy) toàn bộ hệ thống (FE, BE) lên môi trường máy chủ AWS EC2
- Cấu hình Web Server (Nginx/Apache) để public website ra Internet		 18/06/2026 18/06/2026
6 - Thực hiện kiểm thử (Manual Testing) các chức năng chính: Thêm sản phẩm, giỏ hàng, đặt hàng
- Kiểm tra tốc độ tải trang và phản hồi của API		 19/06/2026 19/06/2026
7 - Sửa lỗi (Fix bug) dựa trên kết quả kiểm thử
- Xác nhận hệ thống hoạt động ổn định và họp nhóm chốt giai đoạn phát triển		 20/06/2026 20/06/2026

Kết quả đạt được tuần 9:

  • Hoàn thiện 100% các chức năng cốt lõi (Quản lý sản phẩm, Giỏ hàng) của website bán kèn Saxophone.
  • Triển khai thử nghiệm thành công toàn bộ hệ thống (Frontend + Backend) lên môi trường AWS EC2 thực tế.
  • Các điểm yếu trong hạ tầng Cloud đã được rà soát và tối ưu hóa bảo mật thành công.
  • Hệ thống vượt qua quá trình kiểm thử, hoạt động ổn định, mượt mà và sẵn sàng bước vào giai đoạn kiểm tra cuối cùng (hoàn thiện tiếp theo).
