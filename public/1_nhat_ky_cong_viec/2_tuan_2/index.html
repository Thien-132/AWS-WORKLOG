1.2. Worklog Tuần 2

Mục tiêu tuần 2:

  • Làm quen với IAM User và các nguyên tắc phân quyền cơ bản.
  • Cài đặt và thực hành các thao tác cơ bản với AWS CLI.
  • Tìm hiểu và thực hành cách kết nối EC2 bằng SSH.
  • Thực hành tạo và quản lý S3 Bucket (Amazon Simple Storage Service).

Các công việc cần triển khai trong tuần này:

Thứ Công việc Ngày bắt đầu Ngày hoàn thành Nguồn tài liệu
2 - Làm quen với AWS IAM (Identity and Access Management)
- Tìm hiểu về các khái niệm cốt lõi: IAM User, Group, Role và Policy		 27/04/2026 27/04/2026 https://docs.aws.amazon.com/iam/
3 - Thực hành IAM:
+ Tạo IAM User và phân quyền (Permissions) cơ bản
+ Tìm hiểu và áp dụng nguyên tắc quyền hạn tối thiểu (Least Privilege)		 28/04/2026 28/04/2026
4 - Cài đặt AWS CLI và cấu hình tài khoản qua Access Key / Secret Key
- Thực hành các lệnh AWS CLI cơ bản (như kiểm tra user, liệt kê danh sách tài nguyên)		 29/04/2026 29/04/2026 https://docs.aws.amazon.com/cli/
5 - Tìm hiểu sâu về các phương pháp kết nối vào EC2 bằng SSH
- Sử dụng key pair (.pem), cấu hình phân quyền file an toàn trên máy tính cá nhân
- Khắc phục một số lỗi SSH thường gặp		 30/04/2026 30/04/2026 https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/AccessingInstances.html
6 - Tìm hiểu dịch vụ lưu trữ Amazon S3
- Các khái niệm quan trọng: Bucket, Object, Storage Classes, Versioning		 01/05/2026 01/05/2026 https://docs.aws.amazon.com/s3/
7 - Thực hành S3:
+ Tạo S3 Bucket bằng giao diện Web Console và qua AWS CLI
+ Upload dữ liệu, quản lý và thiết lập quyền truy cập (Public/Private) của Object		 02/05/2026 02/05/2026

Kết quả đạt được tuần 2:

  • Nắm vững kiến thức bảo mật cơ bản với AWS IAM và nguyên tắc Least Privilege.
  • Đã tự tạo IAM User để sử dụng thay vì dùng tài khoản Root cho các tác vụ hàng ngày.
  • Thành thạo cách cài đặt, cấu hình và sử dụng AWS CLI để thao tác với tài nguyên nhanh chóng thông qua Terminal.
  • Hiểu rõ cơ chế bảo mật bằng Key Pair và kết nối thành công tới EC2 instance thông qua giao thức SSH một cách an toàn.
  • Thực hành Amazon S3:
    • Thao tác tạo S3 Bucket thành công bằng cả hai cách: Console và CLI.
    • Nắm được cách thiết lập quyền truy cập an toàn cho bucket và dữ liệu lưu trữ.
    • Thực hành upload, download object trơn tru.
